About This Tool

Care Attend uses machine learning to predict which patients are at risk of missing their appointment. Predictions are explained using SHAP (SHapley Additive exPlanations), and the system monitors for demographic bias.

Data Handling: No patient data is stored. All information is processed in-session only and cleared when you close the browser. Fully compliant with UK GDPR Article 5(1)(c) data minimisation principles.

Appointment Slot Optimisation

Analyse slots to identify overbooking opportunities based on DNA risk. Paste JSON array of patient records.

Optimisation Summary

Slot-by-Slot Analysis

How It Works

Slots with ≥40% DNA probability flagged as overbookable. Expected waste = DNA probability × slot duration.

NHS Impact: If 15% of slots are overbookable, a practice running 200 appointments/week could recover ~30 slots (7.5 clinical hours/week).

About Patient Nudge

Messages auto-personalised based on risk factors: age, deprivation, disability, DNA history. System selects appropriate type (transport, social support, gentle reminder) without stigmatising.

Multi-language: English, Welsh, Urdu, Polish — covering 4.2M UK residents who don't speak English as primary language.
